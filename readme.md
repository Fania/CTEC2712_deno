# Readme


**Enable Deno:**

Open Command Palette and find `Deno: enable` command.

Run Deno with `deno --allow-net --allow-read main.js`


Part 1: a basic Deno server

Part 2: static files

Add package/library: `deno add jsr:@std/http`

Part 3: MVC structure and CSS

Part 4: Models and SQLite

Add package/library: `deno add jsr:@db/sqlite`

Run Deno: `deno task serve`

Add SQLite extension in VSCode.
Open Command Palette and find `SQLite open DB` command.
